Etiquetas Formularios

Etiqueta form

la etiqueta form sirve para representar un formulario


Etiqueta label

La etiqueta label sirve para representar el titulo de un formulario


Etiqueta textarea

La etiqueta textarea sirve para representar un control de edición de texto multilínea.


Etiqueta keygen

La etiqueta keygen sirve para representar el control de un par generador de llaves


Etiqueta output

La etiqueta output sirve para representa el resultado de un cálculo.
